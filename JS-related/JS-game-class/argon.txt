아르곤의 구성요소

1. 플레이어
 - 0, 1, 2, 3레벨. 0은 기본총알 1개씩 한 줄, 1은 대형총알 1개씩 한 줄, 2는 기본 3개, 3은 기본총알 5개
	이동 4방향(대각선 가능), 발사키 1개
	캔버스 안에서만 이동 가능함

2. 발사체
 - 레벨 0, 2, 3 플레이어 총알
 - 레벨 1 총알

3. 적
 - 이동형 적
	사이즈와 색상이 다양함. 어두운 갈색이나 완전 하양은 없는걸로 봐선 고정된 값이 아닐까...)
	체력은 같은것으로 보이지만 크기때문인지 검증이 어려움.
 - 벽 적
	사이즈는 전부 균일한걸로 보이고 색상은 각 웨이브마다 동일함. 색상 선택은 이동형과 마찬가지

4. 배경
 - 검은색

5. 파티클
 - 적 파괴 파티클(적의 색에서 어두워지며 크기 작아짐/폭발 지점에서 멀어져감)


 - 플레이어 파괴(본 예제에는 플레이어 체력 시스템이 없는듯)
